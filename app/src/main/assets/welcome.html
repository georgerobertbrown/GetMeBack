Welcome to GetMeBack

Overview

This app allows you to mark a location and then easily get directions to return. This is useful for example if you park your car and go shopping and then forget where your car is.

Location

GetMeBack uses your phone's location services as well as Google Map's turn-by-turn directions to return to the marked location. Navigation modes are:
  1. Driving
  2. Walking
When the app is launched the first time, it will request location permission. Precise location mode must be selected.

Home screen widget

A Home screen widget can be created to quickly set the current location and return to the destination. Steps:
  1. long-press on the Home screen
  2. select Widgets
  3. click GetMeBack
  4. press and hold the widget preview icon
  5. drag the widget to the Home screen
